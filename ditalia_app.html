App Pizzaria D'Italia

Aplicação multi-plataforma para realização de pedidos.

Este projeto foi criado com o intuito de facilitar a realização de pedidos pelos clientes durante a pandemia do codiv-19 ocorrida em 2020.

O projeto contou com o desenvolvimento de um aplicativo multi-plataforma (Flutter) e uma api para controle dos dados (Node.js), bem como sua implatanação e manutenção.

O aplicativo

O aplicativo, desenvolvido em Flutter para as plataformas Android e iOS, permite ao usuário navegar pelo cardápio do estabelecimento, selecionando seus itens e customizando de acordo com o tamanho, sabores e adicionais a gosto do cliente.

Sob medida

O cliente pode acompanhar seus pedidos, cadastrar e alterar seus dados pessoais e endereços, bem como informar se deseja apenas pedir pelo app e retirar no local.

No app é informada a forma de pagamento e dados como quantia que irá pagar e o troco e o pagamento é feito no ato da entrega.

A API

A API do aplicativo, que foi desenvolvida em Node.js, realiza todo o processamento dos pedidos, bem como realiza a listagem de todos os dados utilizados pelo app, como categorias, cardápio, pedidos, formas de pagamento, mensagens e etc. A API também conta com funções de integração e auditoria de pedidos.

A pedido do cliente, a API também controla as impressões dos pedidos nas impressoras do estavbelecimento de forma automática. Assim que chega um pedido, cada setor fica sabendo dele por meio de sua impressora térmica.

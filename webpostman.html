Web Postman

Web app que permite teste rápido de requisições HTTP.

Este projeto foi criado com o intuito de aprender e aprimorar técnicas de utilização das funções javascript fornecidas pelos navegadores para realizar requisições GET e POST simples. Como resultado temos uma aplicação que permite testes rápidos destas requisições.

Vale salientar que este web app está sucetível a bloqueios de acesso aos dados de retorno pelo navegador, como por exemplo pode acontecer caso seu endpoint não aceite requisições CORS.

Caso possua alguma sugestão de melhoria ou crítica, sinta-se livre para requisitar uma alteração ou abrir uma issue no repositório do projeto no Github.

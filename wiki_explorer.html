Wikipedia Explorer

App Android/iOS que permite navegar e favoritar seus artigos preferidos do Wikipedia.

Este é um projeto criado com o intuito de aprimorar técnicas utilizando o Flutter, em especial o uso do componente WebView para acesso e manipulação de dados de páginas web.

O projeto foi pensado para contar com processos de controle de navegação do usuário, manipulação dos dados da página carregada e armazenamento dos artigos, utilizando o SQLite com o pacote sqflite.

Leitura descomplicada

O Wikipedia Explorer permite ao usuário navegar por toda a Wikipedia, contando com controles de navegação próprios.

Os artigos podem ser favoritados para que sejam acessados com facilidade, pela aba de Favoritos.

Salve para depois

Para salvar os dados foi criado um banco de dados local, utilizando o SQLite com auxiliio do pacote sqflite.

O usuário pode navegar por toda a Wikipedia, porém o app conta com um controle de navegação, analisando sempre se o usuário está ou não acessando um sub-domínio da Wikipedia. Caso tente acessar um link para um site externo, o mesmo será aberto no navegador.

ConCEP

Busque informações de um CEP de forma prática e rápida

Em diversos casos precisamos checar ou obter informações sobre uma localização, como nomes de ruas, cidade, estado, logradouro, ou até mesmo o código do IBGE da cidade, usando um CEP. Visando aprender a utilizar a API ViaCEP e também a implementá-la numa aplicação, desenvolvi o ConCEP.

Este web-app pode ser acessado via navegador por qualquer dispositivo e permite que você faça consultas por CEP, retornando as informações correspondentes (se houver) cadastradas na base de dados da ViaCEP.

O ConCEP é um projeto de código aberto com repositório no GitHub e hospedado via GitHub Pages.

Salve os resultados da pesquisa

O ConCEP permite que o usuário salve os dados da pesquisa, quando encontrados, no formato de texto (.txt) e no formato JSON (.json). Ambas as opções são acessíveis no canto superior esquerdo do card de informações da pesquisa.
